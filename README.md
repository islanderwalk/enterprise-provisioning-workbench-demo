# Enterprise Provisioning Workbench Demo

Sanitized public showcase for a desktop automation workflow that handles:

- batch request extraction
- deduplication and merge policy
- downstream account provisioning stages
- exception-focused review
- mail draft generation

## What This Public Repo Shows

- architecture and workflow thinking
- batch-first UI direction
- deterministic adapter design
- safe testable abstractions

## What It Does Not Expose

- internal company systems
- real target URLs
- real credentials
- raw HTML captures
- production debug bundles

## Suggested Public Repo Structure

- `docs/architecture-overview.md`
- `docs/interview-story.md`
- `docs/sanitization-checklist.md`
- `src/` or `demo_app/` for sanitized implementation
- `tests/` for public-safe test coverage

Use this folder as the seed for a separate public GitHub repository, not as a nested production repo.
